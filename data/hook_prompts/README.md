# Hook Prompts Per Account

Each account has its own folder:
- `data/hook_prompts/account_01/prompt.txt`
- `data/hook_prompts/account_02/prompt.txt`
- `data/hook_prompts/account_03/prompt.txt`
- `data/hook_prompts/account_04/prompt.txt`
- `data/hook_prompts/account_05/prompt.txt`

Edit each `prompt.txt` to control viral hook style for that specific account.
This text is automatically injected into `/generate-viral-hooks` generation.
